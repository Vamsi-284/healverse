HealVerse is a healthcare management system designed to streamline hospital and clinic operations.  
It helps manage patients, doctors, appointments, and medical records efficiently with a centralized platform.

Project Goal
To build a real-world full-stack healthcare system using Java and SQL that demonstrates backend logic, database design, and system architecture skills.

Core Features
- Patient Registration & Management
- Doctor Management
- Appointment Scheduling
- Electronic Medical Records (EMR)
- Admin Dashboard
- Search & Filter System

Advanced Features (Planned)
- Analytics Dashboard (appointments & patient trends)
- Role-based Login (Admin / Doctor / Receptionist)
- Notification System
- Report Generation

Tech Stack
- Language: Java
- Database: MySQL
- Backend: JDBC / Spring Boot (later)
- Version Control: Git & GitHub

Project Status
- Planning Phase (Architecture & Database Design)
